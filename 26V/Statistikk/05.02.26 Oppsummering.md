### Tre lover til Kolmogorov (Axioms)
1. $O <= P(A) <= 1$
2. $P(\ohm) = 1$
3. Hvis $AB = \varnothing$ (Disjoint), then $P(A\cup B) = P(A) + P(B)$

- Ekstra lov
1. $P(A^c) = 1-P(A\cup B)$ ; Sannsynlighet til komplimentet.

### Ex: "Bursdagsparadokset"
2 stk: $P(A_{2}^{c}=\frac{1}{365}, P(A_{2} = \frac{364}{365}))$
$P(A_{2}^{c})=1-P(A_{2})$
3 stk: $P(A_{3})=\frac{364}{365}\cdot\frac{363}{365}$ 
4 stk: $P(A_{4})=\frac{364}{365}\cdot\frac{363}{365}\cdot\frac{362}{365}$
5 stk: $P(A_{5})=\frac{364}{365}\cdot\frac{363}{365}\cdot\frac{362}{365}\cdot\frac{361}{365}$
6 stk: $P(A_{5})=\frac{364}{365}\cdot\frac{363}{365}\cdot\frac{362}{365}\cdot\frac{361}{365}\cdot\frac{360}{365}$
$P(A_{n}^c)=1-P(A_{n})$

Når (hvilken n) er $P(A_{n}^c) \geq \frac{1}{2}$ blir det for høyt.

```R
pan = function(n) {
	brok = 1
	if (n>1)
		for (k in 2:n)
		{
			brok = brok * (365-k+1)/365
		}
	}
	brok
}
```
